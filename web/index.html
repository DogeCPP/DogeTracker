Open on other devices

The plugin is now accessible from any device on the same WiFi or local network.

On your phone, tablet, or another PC open:

http://<xplane-pc-ip>:4000

Find the IP in the X-Plane developer console log — look for the line that starts with Network :

On Windows you can also run ipconfig in Command Prompt and look for your IPv4 Address under Wi-Fi or Ethernet.

Note: Windows Firewall may block the first connection. If the page doesn't load on other devices, add an inbound rule for TCP port 4000 in Windows Defender Firewall.
